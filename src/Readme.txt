
. All files in "data" directory needs to be in the programs directory when executing the program

. An overview of the commands is below
No command line argument is needed to run the file.

Note:Esc is used to exit the program

1) Perseputal user interface by hand and fingertip tracking through AR objects.
run => hand.py 

2) Place your hand infront of  the camera to detect hand.

3) The finger tip count is done and is used as an input to the cube to move on the image plane.

3) Camera Calibration is already done automatically using the chessboard image.

4) The 3 D cube is projected on the chessboard imgae.

5) The input to the cube is given by the Finger Tip count.
